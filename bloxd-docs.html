Callbacks & Events

Daftar callbacks yang bisa dipakai di BloxdCore. Gunakan ini untuk memicu script ketika suatu event terjadi:

# Player Events onPlayerJoin(player) # Trigger saat pemain join onPlayerLeave(player) # Trigger saat pemain keluar onPlayerChat(player, message) # Trigger saat pemain mengirim chat onPlayerMove(player, fromPos, toPos) # Trigger saat pemain bergerak onPlayerRespawn(player) # Trigger saat pemain respawn onPlayerDeath(player, killer) # Trigger saat pemain mati # Block Events onBlockPlace(player, block) # Trigger saat block ditempatkan onBlockBreak(player, block) # Trigger saat block dihancurkan onBlockInteract(player, block) # Trigger saat pemain interaksi dengan block onBlockUpdate(block) # Trigger saat block berubah status onBlockTick(block) # Trigger setiap tick untuk block tertentu # Entity Events onEntitySpawn(entity) # Trigger saat entity muncul onEntityDeath(entity, killer) # Trigger saat entity mati onEntityInteract(player, entity) # Trigger saat pemain interaksi dengan entity onEntityDamage(entity, amount, attacker)# Trigger saat entity menerima damage # Item & Inventory Events onItemUse(player, item) # Trigger saat pemain pakai item onItemPickup(player, item) # Trigger saat pemain mengambil item onItemDrop(player, item) # Trigger saat pemain menjatuhkan item onInventoryOpen(player, inventory) # Trigger saat inventory dibuka onInventoryClose(player, inventory) # Trigger saat inventory ditutup # Game Events onGameStart() # Trigger saat game mulai onGameEnd() # Trigger saat game berakhir onCheckpointReached(player, checkpoint) # Trigger saat pemain mencapai checkpoint onGoalReached(player, goal) # Trigger saat pemain mencapai goal onTimerTick(timer) # Trigger setiap tick timer tertentu # Custom Events triggerCustomEvent(name, data) # Memicu custom event sendiri listenCustomEvent(name, callback) # Menangkap custom event # Miscellaneous onExplosion(location, radius, source) # Trigger saat terjadi ledakan onWeatherChange(newWeather) # Trigger saat cuaca berubah onDayNightChange(newTime) # Trigger saat pergantian siang/malam onServerShutdown() # Trigger saat server mati
