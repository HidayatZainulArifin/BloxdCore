Callbacks & Events

Daftar callbacks yang bisa dipakai di BloxdCore. Gunakan ini untuk memicu script ketika suatu event terjadi:

# Player Events onPlayerJoin(player) onPlayerLeave(player) onPlayerChat(player, message) onPlayerMove(player, fromPos, toPos) onPlayerRespawn(player) onPlayerDeath(player, killer) # Block Events onBlockPlace(player, block) onBlockBreak(player, block) onBlockInteract(player, block) onBlockUpdate(block) onBlockTick(block) # Entity Events onEntitySpawn(entity) onEntityDeath(entity, killer) onEntityInteract(player, entity) onEntityDamage(entity, amount, attacker) # Item & Inventory Events onItemUse(player, item) onItemPickup(player, item) onItemDrop(player, item) onInventoryOpen(player, inventory) onInventoryClose(player, inventory) # Game Events onGameStart() onGameEnd() onCheckpointReached(player, checkpoint) onGoalReached(player, goal) onTimerTick(timer) # Custom Events triggerCustomEvent(name, data) listenCustomEvent(name, callback) # Miscellaneous onExplosion(location, radius, source) onWeatherChange(newWeather) onDayNightChange(newTime) onServerShutdown()

Scripts & Event Examples

Contoh penggunaan callbacks di script BloxdCore:

# Contoh: Memunculkan pesan saat pemain join onPlayerJoin(player): sendMessage(player, "Selamat datang di server!") # Contoh: Ledakan saat block dihancurkan onBlockBreak(player, block): if block.name == "Moonstone Ore": createExplosion(block.position, 5) # Contoh: Custom event triggerCustomEvent("ShowFireworks", {player: player}) listenCustomEvent("ShowFireworks", function(data): spawnFirework(data.player.position)
