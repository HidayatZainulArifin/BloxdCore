Technical Documentation

Player Callbacks (F8 Events)

Assign fungsi ini di World Code lu untuk mendeteksi aksi pemain:

onPlayerJoin = (playerId) => { ... }
onPlayerChat = (playerId, chatMessage) => { ... }
onBlockBreak = (playerId, x, y, z, blockName) => { ... }
onPlayerDeath = (killedPlayer, attackingLifeform) => { ... }

Client Options

Gunakan api.setClientOption(playerId, option, value)

canChange (bool)
speedMultiplier (num) - Default: 1
jumpAmount (num) - Default: 8
maxHealth (num) - Default: 100
invincible (bool)
creative (bool)
fallDamage (bool)
bounciness (num)
