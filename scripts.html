BloxdCore Scripts & Event Examples

Player Scripts

Contoh script memanfaatkan player callbacks:

# Pesan sambutan saat player join onPlayerJoin(player): sendMessage(player, "Selamat datang di server BloxdCore!") # Respawn player dengan efek partikel onPlayerRespawn(player): spawnParticles(player.position, "spark", 30) teleport(player, getSpawnPoint())

Block Scripts

Script contoh untuk block events:

# Ledakan saat Moonstone Ore dihancurkan onBlockBreak(player, block): if block.name == "Moonstone Ore": createExplosion(block.position, 5) # Auto-grow tanaman onBlockTick(block): if block.name == "Wheat_stage1": setBlock(block.position, "Wheat_stage2")

Custom Events

Contoh penggunaan custom events:

# Memicu fireworks untuk player tertentu triggerCustomEvent("ShowFireworks", {player: player}) # Menangkap custom event listenCustomEvent("ShowFireworks", function(data): spawnFirework(data.player.position)

Misc Scripts

Lain-lain:

# Mengubah cuaca onWeatherChange(newWeather): if newWeather == "rain": sendMessageToAll("Hujan turun, bawa payung!") # Day/Night transition onDayNightChange(newTime): if newTime == "night": spawnNightMobs() else: despawnNightMobs()
