Ready-to-Use Scripts

Salin kode ini ke menu F8 (World Code) atau Code Block lu.

1. Auto Welcome & Color Chat

onPlayerJoin = (playerId) => { api.sendMessage(playerId, "Selamat Datang di BloxdCore World!", { color: "aqua" }); }

2. Magic Jump Boost (Command !jump)

onPlayerChat = (playerId, message) => { if (message === "!jump") { api.setVelocity(playerId, 0, 15, 0); api.sendMessage(playerId, "BOOOM! Tinggi banget!"); } }

3. Anti-Void Protection

tick = () => { for (const id of api.getPlayerIds()) { const pos = api.getPosition(id); if (pos[1] < -10) { api.setPosition(id, 0, 5, 0); api.sendMessage(id, "Selamat! Lu hampir jatuh ke void."); } } }
